The uniqueness of projection operator¶

set environment

[1]:

import numpy as np

Define an example¶

[2]:

X = np.array([
    [1, 1],
    [1, 2],
    [1, 3],
    [1, 4]])
print(X)

[[1 1]
 [1 2]
 [1 3]
 [1 4]]

Create first Projection matrix¶

QR decomposition to get a orthonormal basis of col(X)

[3]:

Qx,R = np.linalg.qr(X)
print(Qx)

[[-0.5         0.67082039]
 [-0.5         0.2236068 ]
 [-0.5        -0.2236068 ]
 [-0.5        -0.67082039]]

Show that it is orthonormal basis

[4]:

u = Qx[:,0]; v = Qx[:,1]
print(u.dot(u))
print(v.dot(v))
print(u.dot(v))

1.0
0.9999999999999998
8.326672684688674e-17

[5]:

Px = Qx.dot(Qx.T)
print(Px)

[[ 0.7  0.4  0.1 -0.2]
 [ 0.4  0.3  0.2  0.1]
 [ 0.1  0.2  0.3  0.4]
 [-0.2  0.1  0.4  0.7]]
